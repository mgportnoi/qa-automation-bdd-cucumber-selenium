# QA Automation BDD - Cucumber + Selenium

Projeto de automação utilizando BDD (Behavior Driven Development) com Cucumber, Selenium WebDriver e Java.

## Objetivo

Demonstrar:

- Escrita de cenários em Gherkin
- Separação entre negócio e implementação
- Step Definitions
- Page Object Model
- Automação E2E
- Testes positivos e negativos

## Tecnologias

- Java
- Maven
- Selenium WebDriver
- Cucumber
- JUnit

## Estrutura

- features: Cenários escritos em Gherkin
- steps: Implementação dos passos
- pages: Page Object Model
- runners: Configuração de execução

## Conceitos Aplicados

- BDD
- Gherkin
- Given / When / Then
- Automação orientada a comportamento
- Testes reutilizáveis

## Execução

Executar via Maven:
mvn test

## Cenários Automatizados

- Login com sucesso
- Login inválido
- Validação de mensagens
- Fluxo positivo e negativo
